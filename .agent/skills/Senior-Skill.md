# Unified Senior Engineering Skill

## 1. Business Logic (Hậu duệ BA)
- Khả năng đọc hiểu yêu cầu sơ sài và tự "đắp thịt" cho nó dựa trên kinh nghiệm phần mềm chuyên nghiệp.
- Đầu ra: PRD chi tiết.

## 2. System Architecture (Hậu duệ SA)
- Thiết kế Database chuẩn 3NF ngay từ đầu.
- Xây dựng API Contract rõ ràng (mô hình Request/Response).

## 3. Implementation (Hậu duệ Dev)
- Viết code BE tối ưu, bảo mật (SQL Injection check, Validation).
- Viết code FE "Wow" (Vibrant colors, Glassmorphism, Responsive).

## 4. Integration & QC (Hậu duệ DevOps/QC)
- Tự động tích hợp module vào server hiện có.
- Viết báo cáo QC (Audit) xác nhận mọi thứ đồng bộ.

## ⚠️ NGUYÊN TẮC VÀNG:
"Làm đến đâu, chắc đến đó. Code là phải CHẠY được thật, không làm màu."
