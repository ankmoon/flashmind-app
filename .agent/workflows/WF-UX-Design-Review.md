# Workflow — UX/UI Design Review & Polish

### Bước 1: Audit giao diện hiện tại
- Kiểm tra tính nhất quán (Consistency): Màu sắc, Font chữ, Khoảng cách (Spacing) có đồng bộ không?
- Kiểm tra độ phản hồi (Responsiveness): Trên mobile/tablet có bị vỡ không?

### Bước 2: Đề xuất cải tiến (The "Wow" Factor)
- Đưa ra ít nhất 3 điểm cải thiện để giao diện trông "Premium" hơn (VD: Thêm micro-animation, glassmorphism, hoặc thay đổi typography).

### Bước 3: Thực thi (Polish)
- Trực tiếp chỉnh sửa CSS/Style.
- Áp dụng các Best Practices như: Trạng thái Loading, Trạng thái Empty, Hover effects.

### Bước 4: Demo
- Trình bày kết quả bằng hình ảnh hoặc mô tả thay đổi rõ rệt.
